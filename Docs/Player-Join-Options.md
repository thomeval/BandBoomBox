Before heading into Song Select, each player can configure a range of personal options on the Player Join screen. These settings are saved to your profile and will persist between sessions.

Use Left and Right on the menu to cycle through the available values for each option. Some options open a separate screen when selected.

## Select Profile

Opens the profile selection screen, where you can load an existing saved profile or continue as a guest. Selecting a profile restores all of your saved settings, your personal best scores, your chosen rival, and your accumulated experience.

## Select Rival

Opens the rival selection screen, where you can choose another player's profile to compete against. See the [Goals and Rivals](Goals-and-Rivals) page for full details on how rivals work.

## Scroll Speed

Controls how fast notes scroll down your Note Highway during gameplay. Higher values mean notes travel faster, giving you less time to react but also more visual space between notes. The default scroll speed is 500.

## Timing Display

Controls what feedback is shown on your Note Highway after each note hit, indicating how accurate your hit was, and optionally whether your timing was early or late.

## Note Labels

Controls the text labels printed on the face of each note button. This is a visual preference and does not affect gameplay. It is generally recommended to set this to match the physical layout of your controller, but this can be overridden if desired. The recommended settings for each controller is as follows:

| Controller | Recommended Setting |
|------------|------------------|
| Xbox (all models) | ABXY |
| Nintendo Switch | BAYX |
| PlayStation (all models) | Symbols |
| 8BitDo | Usually BAYX, but varies by model |
| Keyboard | WASD, when using default controls |
| Others | Arrows or None |

> **NOTE:** The game will attempt to automatically detect your controller type and set the Note Labels option to the recommended setting for that controller, but this will fail for unrecognized controllers, or when using controller remapping software such as Steam Input. In these cases, it will be necessary to override the chosen setting manually.

## Goal

Sets a target grade to aim for during a song.

See the [Goals and Rivals](Goals-and-Rivals) page for full details on how goals work, including the EXP bonus table.

## Mistake Sfx

Controls whether an audible sound effect is played when you press the wrong button during gameplay (in other words, `WRONG` judgements). Set this to Off if you prefer a quieter experience or find the sound distracting.

- On (default) - A sound effect plays on each miss or wrong press.
- Off - No sound effect is played on mistakes.

> **NOTE:** To change the volume of the mistake sound effect without disabling it entirely, adjust the "Mistake Sfx Volume" option in the Audio Options menu.

## Controller Rumble

Controls whether your controller will vibrate (rumble) during gameplay. This option only applies when using a gamepad that supports haptic feedback; it has no effect when using a keyboard.

- On (default) - The controller rumbles on relevant gameplay events (such as hitting incorrect buttons during gameplay).
- Off - Controller rumble is disabled.

## Auto Turbo

Controls whether Turbo activates automatically once your Energy Meter is completely full, rather than requiring you to press the Turbo button manually. This option is enabled by default for new profiles and guests. 

See the [Turbo](Turbo) page for more information on how Turbo works.

> **NOTE:** Even with Auto Turbo enabled, you can still press the Turbo button at any time to activate or deactivate it manually.

## Ally Boosts

This controls how you participate in the Ally Boost cooperative mechanic during multiplayer sessions. This option has no effect in single-player games.

See the [Ally Boosts](Ally-Boosts) page for full details on how Ally Boost works.

## Lane Order

Controls the vertical arrangement of the three note lanes (Triggers, Arrows, and ABXY buttons) on your Note Highway. This is a visual preference and does not affect which physical buttons you press.

- Standard (default) - Triggers at the top, Arrows in the middle, ABXY at the bottom.
- BottomTrigger - Arrows at the top, ABXY in the middle, Triggers at the bottom.
- MiddleTrigger - Arrows at the top, Triggers in the middle, ABXY at the bottom.
- Flipped - Triggers at the top, ABXY in the middle, Arrows at the bottom.

## Unlockable Options

The following options are hidden by default and must be unlocked by entering a secret code on the Title Screen. Once unlocked, the setting is saved permanently and will appear automatically in future sessions.

### Momentum Speed

This option increases the note scroll speed dynamically based on the current score multiplier. At 0, your scroll speed stays constant throughout the song. Higher values cause notes to speed up noticeably during high-multiplier stretches, which can be a fun challenge for experienced players.

### Section Difficulty

Controls the thresholds used to grade your performance at the end of each song section, which affects section bonuses. A stricter setting requires a higher hit accuracy to achieve the same section grades.

- Off — Section bonuses are disabled entirely.
- Normal (default) — Standard grade thresholds are used (90% = "Superb", etc).
- Strict — Stricter grade thresholds are used (95% = "Superb", etc), making it more difficult to earn section bonuses.
